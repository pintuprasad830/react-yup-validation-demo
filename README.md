# React Yup Validation Demo

This project demonstrates form validation in React using Formik and Yup.

## Features
- Formik Form Handling
- Yup Validation Schema
- Username Validation
- Age Validation
- Mobile Validation
- City Dropdown Validation
- Gender Validation
- Error Messages

## Technologies Used
- React JS
- Formik
- Yup
- JavaScript
- Bootstrap

## Concepts Covered
- Form Validation
- Yup Schema
- React Hooks
- Error Handling
- Form Submission
